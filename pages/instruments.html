¿Cómo usar el teclado de piano virtual?


¡Bienvenido a tu teclado de piano virtual! Puedes usarlo de dos maneras: con el teclado de tu ordenador o con el ratón. ¡Experimenta! Usa ambas formas (teclado y ratón) para ver cuál se adapta mejor a ti.

A continuación, te explicamos cómo hacerlo:

Usar el teclado del ordenador

Primero debes familiarizarte con las teclas. Cada botón del teclado de tu ordenador está vinculado a una tecla del piano virtual.

Teclas blancas: Están asignadas a las letras de la fila central de tu teclado. Ejemplo: Q, W, E, R, T, Y, U, etc. Teclas negras: Corresponden a los números (notas sostenidas) o algunas letras de las filas superiores. Ejemplo: 2, 3, 5, 6, 7, etc.

Pasos:

  • Identifica las teclas blancas y negras en la pantalla y sus correspondientes en tu teclado físico.
  • Presiona la tecla correspondiente para tocar la nota deseada.
  • Experimenta y crea melodías mientras aprendes la posición de las teclas.

Puedes probar a realizar esta escala simple:
Comienza con la escala de Do mayor (Do, Re, Mi, Fa, Sol, La, Si, Do). Esto significa tocar las teclas en el siguiente orden: Q, W, E, R, T, Y, U, Z.

Usar el ratón

Si prefieres usar el ratón, puedes interactuar directamente con el teclado de la pantalla y para tener más control, puedes mantener el cursor cerca de donde se unen las teclas negras y blancas.

Pasos:

  • Haz clic en las teclas blancas o negras del piano virtual para producir sonidos.
  • Puedes hacer clic en varias teclas de forma rápida para tocar acordes o melodías.
